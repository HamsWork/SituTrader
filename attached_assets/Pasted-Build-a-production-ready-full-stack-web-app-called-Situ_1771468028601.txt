CORE SETUPS TO IMPLEMENT (Minimum Viable):
Setup A — Thu–Fri–Mon Magnet (Hougaard-style)
	•	Evaluate after Friday daily close:
Condition: High(Fri) < High(Thu)
Magnet: Low(Fri)
Target day: next trading day after Friday (normally Monday)
Validation: On target day during RTH, did intraday Low <= Magnet?

Setup B — Mon–Wed–Thu Magnet
	•	Evaluate after Wednesday daily close:
Condition: High(Wed) < High(Mon)
Magnet: Low(Wed)
Target day: next trading day after Wednesday (normally Thursday)
Validation: On target day during RTH, did intraday Low <= Magnet?

Setup C — Gap Fill Magnet
	•	Evaluate at the open (requires intraday):
Define Gap% = (Open(today) - Close(prevDay)) / Close(prevDay)
If abs(Gap%) >= user threshold (default 0.30%):
Magnet: Close(prevDay)
Target day: same day (today)
Validation: During RTH, did price touch magnet? (for gap up: intraday Low <= magnet; for gap down: intraday High >= magnet)

Setup D — Inside Day Expansion + Sweep Watch
	•	Evaluate after daily close:
Condition: High(today) < High(yesterday) AND Low(today) > Low(yesterday)
Targets (magnets) for next day: Yesterday High and Yesterday Low
Validation: During next day RTH, record whether either side was swept first and whether both were touched.

EXTRA “SITUATIONAL” MODULES (Add now, but keep them toggleable):
Module E — PDH/PDL Sweep
	•	Every day, track Previous Day High and Previous Day Low as magnets for the next day.
	•	Validation: Next day RTH touches PDH or PDL (intraday High >= PDH, intraday Low <= PDL).
	•	Record which hit first and whether a reversal occurred after sweep (simple heuristic: after hitting, did price move back at least X ATR% or X% within RTH).

Module F — “Weak Extreme” / Unfinished Auction Proxy (No Market Profile needed)
	•	Proxy definition:
If day’s low is within 0.05% of a mechanical level (PDL, round number .00/.50, or premarket low) AND the day closes in the top 35% of its daily range (strong rejection),
then mark that low as a “weak/unfinished” candidate likely to be revisited within the next 1–3 trading days.
	•	Validation: Check next 3 trading days RTH for touch of that low.

DEFAULT TRADE PLAN RULES (Make configurable in Settings):
	•	If current price is above magnet => SELL bias “down-to-magnet”
	•	If current price is below magnet => BUY bias “up-to-magnet”
Entry trigger options (user-selectable, default = conservative):

	1.	Conservative: Enter only after a 5-minute close breaks in the direction of the magnet AND retests that break level without reclaiming.
	2.	Aggressive: Enter on the first 5-minute close in the direction of the magnet after 09:35 ET.
Invalidation/Stop (user-selectable, default = ATR-based):

	•	Compute ATR(14) on daily bars.
	•	Stop distance default = max(0.25 * ATR, 0.15% of price).
Targets:
	•	T1 = magnet (primary objective)
	•	T2 optional = sweep-through buffer = magnet ± 0.15 * ATR (directional)
	•	Always calculate and display the implied R:R for the selected entry/stop.

CONFIDENCE SCORING (Simple, deterministic, explainable):
Start base = 0.50
Adjustments (clamp 0.05–0.95):
	•	+0.15 if “clean edge”: trigger-day condition margin >= 0.10% of price (e.g., ThuHigh - FriHigh >= 0.10%*Close)
	•	+0.10 if trigger-day volume > 20-day average volume
	•	+0.05 if VIX proxy filter (optional): if SPY realized range (daily True Range) is above its 20d average (more movement favors magnets)
	•	-0.10 if the magnet is extremely far: distance from last close to magnet > 1.25 * ATR
Show the scoring breakdown in the UI so it’s auditable.
