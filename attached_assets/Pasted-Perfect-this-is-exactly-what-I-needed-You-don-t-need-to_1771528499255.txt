3) What to compute (derived metrics)

You need these for each ACTIVE signal:

Let:
	•	E = entryPriceAtActivation ?? entryTriggerPrice (but for TradeNow, should exist)
	•	S = stopPrice
	•	T = magnetPrice (or tp.t1 if same thing)
	•	P = currentPrice
	•	direction: BUY or SELL

Progress to target (0–1)

BUY:
	•	progress = (P - E) / (T - E)
SELL:
	•	progress = (E - P) / (E - T)
Clamp 0..1.

R Now

BUY:
	•	rNow = (P - E) / (E - S)
SELL:
	•	rNow = (E - P) / (S - E)

Distance to target + stop (in ATR)

If you have ATR:
	•	distToTargetAtr = abs(T - P) / atr14
	•	distToStopAtr = abs(P - S) / atr14
